<h3>ปุ่มควบคุม</h3>

| ปุ่ม | ในเมนู | ระหว่างเล่น |
| --- | --- | --- |
| `↑` | เลื่อนขึ้น | กระโดด/ขึ้น |
| `←` | เลื่อนซ้าย | เดินซ้าย |
| `→` | เลื่อนขวา | เดินขวา |
| `↓` | เลื่อนลง | คลาน/ลง |
| `Space` | ยืนยัน/คุย/เช็ค | วิ่ง |
| `Z` | ยืนยัน/คุย/เช็ค | โจมตี |
| `X` | ยกเลิก/ข้าม | สกิลเวท |
| `C` | - | เปลี่ยนเป้าหมาย |
| `A` | - | แมพ |
| `S` | - | อื่นๆ |
| `D` | - | ไอเทม |
| `Q` | - | แท็บซ้าย |
| `E` | - | แท็บขวา |
| `LShift` | - | หลบ/ใช้เกราะ |
| `Esc` | 	ยกเลิก | เมนู |
| กด `X` ค้าง | - | ชาร์จเวท |
| `Z` + `X` | - | ยกเลิกสกิลเวท |
| `↓` + `←` หรือ `→` | - | คลานซ้าย/ขวา |
| กด `C` ค้าง | - | หยุด Reel เร็ว |
